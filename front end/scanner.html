About the Webapp

This project is a web application that allows users to scan barcodes of grocery items and retrieve information such as item names and prices from an Excel or SQLite inventory. The application uses a pre-trained MobileNetV2 model to detect barcodes in images uploaded by the user and pyzbar to decode the barcodes once detected.


This Project was created by the Business Affairs Committee on the Wentworth Student Government to make price information of Tessie's More accessible.
